Navegación en el map utilizando teclado y Ratón:
  • Arrastrar para Paneo
  • SHIFT + Click para centrar
  • SHIFT + y arrastrar para Acercar
  • SHIFT + CTRL + y arrastre para alejar
  • Rueda del Rat\u00f3n Adelante para Acercar
  • Rueda del Rat\u00f3n Atras para Alejar
  • Use las teclas de direccion para panear
  • + key to zoom in a level
  • - key to zoom out a level
  • Doble Click para centrar y Acercar
A variety of searches can be performed in the search box:
  • Buscar por Dirección
  • Buscar por Nombre del Sitio
  • Buscar por Dirección postal,Pais, etc.
En adici\u00f3n con con las capacidades de Busqueda, las siguientes herramientas son suministradas
  • Medici\u00f3n
    • La herramienta de medida proporciona la capacidad para dibujar un punto, línea o polígono en el mapa y especificar la unidad de medida.
  • Impresión
    • Este mapa puede se exportado a varios diseños y formatos.
